===
ABS
===

Overview
--------

Returns the absolute value of a number.

Syntax
------

``=ABS(Number)``

Number is the number whose absolute value is to be calculated. The absolute value of a number is its value without the +/- sign.

Example
-------

``=ABS(-56)`` returns 56.

``=ABS(12)`` returns 12.

``=ABS(0)`` returns 0.
