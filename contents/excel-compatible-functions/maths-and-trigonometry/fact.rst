====
FACT
====

Overview
--------

Returns the factorial of a number.

Syntax
------

``=FACT(Number)``

Returns Number!, the factorial of Number, calculated as 1*2*3*4* ... * Number.

``=FACT(0)`` returns 1 by definition.

The factorial of a negative number returns the #NUM! error.

Example
-------

=FACT(3) returns 6.

=FACT(0) returns 1.
