====
EVEN
====

Overview
--------

Rounds a positive number up to the next even integer and a negative number down to the next even integer.

Syntax
------

``=EVEN(Number)``

Returns Number rounded to the next even integer up, away from zero.

Examples
--------

``=EVEN(2.3)`` returns 4.

``=EVEN(2)`` returns 2.

``=EVEN(0)`` returns 0.

``=EVEN(-0.5)`` returns -2.
