===
ODD
===

Overview
--------

Rounds a positive number up to the nearest odd integer and a negative number down to the nearest odd integer.

Syntax
------

``=ODD(Number)``

Returns Number rounded to the next odd integer up, away from zero.

Example
-------

``=ODD(1.2)`` returns 3.

``=ODD(1)`` returns 1.

``=ODD(0)`` returns 1.

``=ODD(-3.1)`` returns -5.
