====
REPT
====

Overview
--------

Repeats a character string by the given number of copies.

Syntax
------

``=REPT("Text", Number)``

.. tabularcolumns:: |l|L|

=============== ================================================================
Arguments       Description
=============== ================================================================
``Text``        The text to be repeated.

``Number``      The number of repetitions.
=============== ================================================================

Example
-------

``=REPT("Good morning",2)`` returns Good morningGood morning.
