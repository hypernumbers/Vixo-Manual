===============
TWITTER.PROFILE
===============

Overview
--------

``TWITTER.PROFILE`` creates a box with the Twitter profile of a Twitter user in it.

Syntax
------

``=twitter.profile(TwitterUser)``

Arguments
---------

.. tabularcolumns:: |l|L|

================= =============================================================
Argument          Description
================= =============================================================
``TwitterUser``   The Twitter Id of the account to be profiled.
