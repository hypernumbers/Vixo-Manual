=====
RADIO
=====

Overview
--------

Creates a set of radio buttons on a webform.

Syntax
------

``=RADIO(Label)``

Arguments
---------

====================  =========================================================
Argument              Description
====================  =========================================================
``Label``             The name of the column under which the data in this input
                      field will be stored.
====================  =========================================================
