=====
INPUT
=====

Overview
--------

The ``INPUT`` function creates an input field as part of a webform.

Syntax
------

``=INPUT(Label)``

Arguments
---------

====================  =========================================================
Argument              Description
====================  =========================================================
``Label``             The name of the column under which the data in this input
                      field will be stored.
====================  =========================================================
