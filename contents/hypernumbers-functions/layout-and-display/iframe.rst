==========
IFRAME.WxH
==========

Overview
--------


The ``iframe.WxH()`` displays another web page inside your page in an iframe.


Syntax
------

``=iframe.WxH(Url)``

Arguments
---------

.. tabularcolumns:: |l|L|

============= ==================================================================
Argument      Description
============= ==================================================================
``W``         Width in cells

``H``         Height in cells

``Url``       A string containing the Url of the page you wish to display.
============= ==================================================================

Example
-------

``=iframe.15x30("http://google.com")``
