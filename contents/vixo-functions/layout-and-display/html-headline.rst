=============
HTML.HEADLINE
=============

Overview
--------

The ``html.headline.WxH()`` function is designed to give more control over headline elements in layouts.


Syntax
------

``=html.headline.WxH(text)``


Arguments
---------

.. tabularcolumns:: |l|L|

=========== ====================================================================
Argument    Description
=========== ====================================================================
``text``    the text of the headline

=========== ====================================================================
