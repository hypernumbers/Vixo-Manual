=====
QUOTE
=====

Overview
--------

Quote takes a string and returns it wrapped in quotes.

Syntax
------

``=quote(Ref)``

.. tabularcolumns:: |l|L|

============= ==================================================================
Arguments     Description
============= ==================================================================
``Ref``       A reference to some text

============= ==================================================================

Example
-------

``=quote(A14)`` returns ``"bingo"`` if ``A14`` contains ``bingo``.
